# Hello World

Simple CPC program, copied from http://www.cpcmania.com/Docs/Programming/Introduction_to_programming_in_SDCC_Compiling_and_testing_a_Hello_World.htm

## Setup

Download dependencies for compiling:

```sh
brew install \
    sdcc \
    clang-format \
    asmfmt
```

## Compile

```sh
make
```
